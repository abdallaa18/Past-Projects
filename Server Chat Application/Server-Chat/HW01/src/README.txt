In order to compile and run this project, please follow the following steps:

1) Download the code and save it into Eclipse (or a software you would like to use)
2) Run Server.java
3) Run as many instances of MyClient.java as needed
4) Once you enter in a username for a client, that client will connect to the server
5) Send messages through the console, messages will appear in the clients console and the servers console
6) Once finished press the red square to quit
